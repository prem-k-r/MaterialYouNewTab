# Extension Locale Metadata

This folder is for browser extension metadata translations used by `chrome.i18n` / WebExtension i18n.

- `locales/` contains the in-app New Tab UI translations.
- `_locales/` contains extension name and description translations for `manifest.json` .
- Each locale folder contains a `messages.json` file.
- Locale folder names use underscores for region variants, for example `zh_CN`, `zh_TW`, and `pt_BR`.

Note: MYNT is pronounced "Mint", not "My-ent".
